PYTHON

Python is a high-level, interpreted programming language that emphasizes readability, simplicity, and ease of use.
It was created by **Guido van Rossum** and first released in **1991**.
Python is known for its straightforward syntax, which makes it a popular choice for beginners and experienced developers alike.
It can be used for a wide range of applications, from web development and data analysis to machine learning and artificial intelligence.

Advantages of Python:

  • Easy to Learn and Read: Python's syntax is simple and intuitive, making it easy to read and write. This makes it a great choice for beginners.
  • Cross-platform Compatibility: Python is platform-independent, meaning you can write Python code on one operating system and run it on another without modification.
  • Extensive Libraries and Frameworks: Python has a large standard library and many third-party libraries and frameworks that make it easy to implement complex functionalities (e.g., NumPy, Pandas, TensorFlow, Django).
  • Open Source: Python is open-source, meaning it's free to use, and there is an active community contributing to its development.
  • Versatility: Python can be used for a wide variety of applications such as web development, automation, data science, artificial intelligence, and more.
  • Large Community: Python has a large and active community, meaning there is a wealth of tutorials, documentation, and forums available to help with troubleshooting and learning.

    • Variables in Python

    In Python, a **variable** is a name that is used to store data. You can think of it as a container where information can be stored for later use. Variables are dynamically typed in Python, meaning you don’t need to declare their type explicitly.

    x = 10 # Here, 'x' is a variable storing an integer value
    name = "John" # 'name' is a variable storing a string

    Data Types in Python:

    Data types define the kind of value a variable can hold. Python supports several built-in data types:

  • Integers (`int`): Whole numbers, positive or negative.

    • x = 5

  • Floating Point Numbers (`float`): Numbers with a decimal point.

    • price = 99.99

  • Strings (`str`): A sequence of characters enclosed in single or double quotes.

    • message = "Hello, Python!"

  • Booleans (`bool`): Represents truth values — `True` or `False`.

    • is_active = True

  • Lists (`list`): An ordered collection of items, which can be of different types.

    • fruits = ["apple", "banana", "cherry"]

  • Tuples (`tuple`): An ordered collection of items, but immutable (can't be changed).

    • coordinates = (10.5, 20.3)

  • Dictionaries (`dict`): A collection of key-value pairs.

    • student = {"name": "Alice", "age": 21}

  • Sets (`set`): An unordered collection of unique items.

    • unique_numbers = {1, 2, 3, 4}

    Operators in Python:

    Operators are symbols that are used to perform operations on variables and values. Python has several types of operators:

  • Arithmetic Operators: Used to perform basic mathematical operations.
    • `+` (Addition)
    `-` (Subtraction)
    `*` (Multiplication)
    `/` (Division)
    `//` (Floor Division)
    `%` (Modulo)
    `**` (Exponentiation)

    x = 10
    y = 5
    sum = x + y # sum is 15

  • Comparison Operators: Used to compare values and return a boolean (`True` or `False`).
    • `==` (Equal to)
    `!=` (Not equal to)
    `<` (Less than)
    `>` (Greater than)
    `>=` (Greater than or equal to)
    `<=` (Less than or equal to)

    x = 10
    y = 5
    result = x > y # result is True

  • Logical Operators: Used to combine conditional statements.
    • `and` (Returns `True` if both conditions are true)
    `or` (Returns `True` if at least one condition is true)
    `not` (Reverses the result, returns `True` if the condition is false)

    x = 10
    y = 5
    result = (x > y) and (y > 0) # result is True

  • Assignment Operators: Used to assign values to variables.
    • `=` (Simple assignment)
    `+=`, `-=`, `*=`, `/=`, etc. (Compound assignment operators)

    x = 5
    x += 2 # x is now 7

  • Identity Operators: Used to compare objects' memory locations.
    • `is` (Returns `True` if both variables point to the same object)
    `is not` (Returns `True` if both variables do not point to the same object)

    a = [1, 2, 3]
    b = a
    result = a is b # result is True

  • Membership Operators: Used to test if a value is present in a sequence (e.g., list, tuple, string).
    • `in` (Returns `True` if the value is present)
    `not in` (Returns `True` if the value is not present)

    fruits = ["apple", "banana", "cherry"]
    result = "apple" in fruits # result is True
